# Branding Assets

Platziere hier deine Branding-Assets für die Video-Erstellung:

## Logo
- `logo.png` - Dein Firmenlogo (transparent, min. 400x400px)

## Intro/Outro Templates (Optional)
- `intro-template.mp4` - Intro-Video (2-3 Sekunden)
- `outro-template.mp4` - Outro-Video (2-3 Sekunden)

## Audio (Optional)
- `background-music.mp3` - Hintergrundmusik (leise, dezent)

## Fonts (Optional)
- Lege hier TTF/OTF Schriftarten ab für custom text overlays

## Beispiele

### Logo erstellen
Exportiere dein Logo als PNG mit transparentem Hintergrund:
- Mindestgröße: 400x400px
- Format: PNG mit Alpha-Kanal
- Empfohlen: Quadratisch oder 16:9

### Intro/Outro erstellen
Kurze Videos (2-3 Sekunden) mit:
- Auflösung: 1920x1080 (Full HD)
- Format: MP4 (H.264)
- Inhalt: Logo-Animation, Titel, Call-to-Action
