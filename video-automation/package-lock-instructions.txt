NPM INSTALL INSTRUCTIONS
========================

Run these commands to install all dependencies:

cd C:\Users\t13016a\Downloads\Team2\storeBackend\video-automation
npm install
npm run install:browsers

This will install:
- Playwright (for browser automation and video recording)
- fluent-ffmpeg (for video processing)
- whisper-node (for subtitle generation)
- dotenv (for environment configuration)

After installation, copy .env.example to .env and configure your settings.
{
  "name": "saas-video-automation",
  "version": "1.0.0",
  "description": "Automated How-to Video Pipeline for SaaS",
  "scripts": {
    "install:browsers": "playwright install chromium",
    "record:all": "node scripts/record-all.js",
    "record:login": "npm run record -- login",

    "record:products": "npm run record -- products",
    "record": "node scripts/record-single.js",
    "process:all": "node scripts/process-all.js",
    "process": "node scripts/process-video.js",
