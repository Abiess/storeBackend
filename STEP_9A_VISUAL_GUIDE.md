## CSS Variables in Action

### Before (Hardcoded)
```scss
.store-header {
  background: white;
  color: #1d1d1f;
}

.btn-primary {
  background: #667eea;
  &:hover {
    background: #764ba2;
  }
}
```

### After (Theme-Aware)
```scss
.store-header {
  background: var(--theme-background, white);
  color: var(--theme-text, #1d1d1f);
}

.btn-primary {
  background: var(--theme-primary, #667eea);
  &:hover {
    background: var(--theme-secondary, #764ba2);
  }
}
```

**Benefits:**
- ✅ No recompilation needed
- ✅ Instant theme changes
- ✅ Fallback values
- ✅ Easy debugging

## DevTools Inspection

**View Applied CSS Variables:**
```
1. Open Storefront
2. F12 → Inspector/Elements
3. Select <html> element
4. Look for Styles → element.style
5. See CSS variables:
   --theme-primary: #667eea;
   --theme-secondary: #764ba2;
   --theme-accent: #f093fb;
   ...
```

**Test CSS Variables:**
```javascript
// In Browser Console:
document.documentElement.style.setProperty('--theme-primary', '#ff0000');
// → All primary-colored elements turn red instantly!
```
